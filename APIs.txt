
This is where front end ppl can call backend functionality


To hit register user: http://(your host)/auth/register with body:
{
  "username": "joeDoe",
  "password": "mypassword123",
  "email": "joe@example.com"
}

To hit login: http://(your host)/auth/login with body:
{
  "username": "joeDoe",
  "password": "mypassword123"
}
